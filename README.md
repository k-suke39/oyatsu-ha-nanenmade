## サービス名

- おやつは何円まで?

## サービス概要

- 駄菓子屋で懐かしさを感じてもらうアプリです。
- 最近セブンでツナマヨのおにぎりが 150 円になってて、高くなったな〜と思いつつも普通に買っちゃったんです。
- 昔だったらそんな大金目の前にしてたら頭フル回転で駄菓子選んでたはずなんです。

## ターゲット層

- 懐かしさを感じたい全ての人へ

## ターゲット層

- 金額に応じた駄菓子レコメンド機能
- Twitter シェア機能
